# Toram_KR
토람온라인 한국어 다중 정보 검색 사이트 실시간 업데이트중...
<br>
[link](toramremote.netlify.app)


# 사용 가능 기능
1. 월간 이벤트 목록
2. 크리스타 검색 (개선중)
3. 스킬 검색 (이미지, 스킬정보 수정중)
4. 장비 어빌리티 검색 (번역 수정중)
5. 레지스트릿 (수정 완료)
6. 요리주소 (추후 추가 수정)
7. 장비 검색 (이미지 정보 추가중)
8. 뉴비가이드 (이미지, 추가적으로 가이드 제작중)
