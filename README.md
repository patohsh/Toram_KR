# Toram_KR
토람온라인 한국어 다중 정보 검색 사이트 실시간 업데이트중...
<br>
[link](https://toramremote.netlify.app/)


# 사용 가능 기능
1. 월간 이벤트 목록
2. 크리스타 검색 (2026.01.15)
3. 스킬 검색 (스킬정보 추가중)
4. 장비 어빌리티 검색 (수정 완료)
5. 레지스트릿 (수정 완료)
6. 요리주소 (추후 추가 수정)
7. 장비 검색 (이미지 정보 추가중)
8. 뉴비가이드 (데이터 베이스형 게시글 작성으로 수정 완료, 이미지 기능 고려중 )
